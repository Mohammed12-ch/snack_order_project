Snack Order est un site web simple qui permet aux utilisateurs de commander des snacks en ligne facilement. Le site affiche un menu contenant différents snacks (sandwichs, boissons, desserts, etc.) avec leurs prix et permet aux utilisateurs de sélectionner les produits qu’ils souhaitent commander.

L’objectif de ce projet est de simuler un système de commande en ligne avec une interface claire et facile à utiliser. Les utilisateurs peuvent parcourir le menu, ajouter des produits au panier et passer une commande.

Ce projet a été réalisé dans le but de pratiquer le développement web front-end et de comprendre comment créer une interface interactive pour un site de commande.

Technologies utilisées :

HTML5 pour la structure du site

CSS3 pour le design et la mise en page

JavaScript pour les fonctionnalités interactives (gestion du panier, interactions utilisateur)

Fonctionnalités principales :

Affichage du menu des snacks

Ajout de produits au panier

Interface utilisateur simple et responsive

Simulation de commande en ligne
